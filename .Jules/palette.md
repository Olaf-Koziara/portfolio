## 2026-02-03 - [Accessibility enhancements for navigation and controls]
**Learning:** Adding a 'Skip to Content' link and proper ARIA labels/states to navigation elements significantly improves the experience for keyboard and screen reader users with minimal code changes.
**Action:** Always check for 'Skip to Content' links in the root layout and ensure toggles/switchers have 'aria-pressed' or 'aria-current' and localized labels.
